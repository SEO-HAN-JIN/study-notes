# Ollama 완벽 정리

## 1. Ollama란?

대형 언어 모델(LLM)을 **로컬 환경에서 쉽게 실행**할 수 있게 해주는 오픈소스 도구.
복잡한 설정 없이 터미널 명령어 하나로 AI 모델을 내 컴퓨터에서 실행할 수 있다.

```bash
ollama run llama3.2  # 모델 다운로드 + 실행이 동시에
```

- **독립 스타트업** (Meta, Google과 무관)

- Meta, Google, Mistral 등 **여러 회사의 모델을 모아서** 쉽게 쓸 수 있게 해주는 역할

> 비유: Meta(영화 제작사) / Llama(영화) / Ollama(플레이어)

## 4. Ollama 구조

```
Ollama 설치 (프로그램만, ~500MB)
        ↓
모델은 별도로 하나씩 다운로드

ollama pull gemma3    # Gemma만 다운
ollama pull llama3.2  # Llama만 다운
ollama pull mistral   # Mistral만 다운
```

- Ollama 설치 시 모든 모델이 포함되는 게 **아님**
- 모델 하나하나가 수 GB이므로, 필요한 것만 골라서 다운로드

Ollama와 모델은 **별개의 라이선스**를 가짐.

| 다운로드 | 라이선스 주인 |

## 6. OpenAI 호환 API

### OpenAI / ChatGPT 관계
```
OpenAI (회사)
    ├── ChatGPT (일반인용 웹/앱 서비스)
    ├── GPT-4, GPT-4o (AI 모델)
    └── OpenAI API (개발자용 API)
```

### OpenAI 호환 API란?
OpenAI가 정해놓은 **요청/응답 형식(규격)** 을 그대로 따르는 것.
ChatGPT를 쓴다는 의미가 아니라, **같은 형식으로 사용 가능하다**는 의미.

```python
# OpenAI SDK를 그대로 사용하면서 주소와 모델명만 변경
client = OpenAI(
    base_url="http://localhost:11434/v1",  # ← 주소만 바꿈
    api_key="ollama"                        # ← 아무 값이나
)
```

> 비유: USB-C 규격처럼, 같은 케이블(코드)로 다른 충전기(모델)를 꽂을 수 있는 것

## 7. 따로 학습이 필요한가?

**아니오.** Ollama에서 받는 모델들은 이미 **수조 개의 데이터로 학습 완료된 상태.**

### 학습 없이 커스터마이징하는 방법
| 방법 | 설명 |
|------|------|
| 프롬프트 지정 | "너는 한국어 고객상담 AI야" 하고 지시 |
| Modelfile | 시스템 프롬프트를 모델에 고정 |
| RAG | 내 문서를 기반으로 답변하게 하기 |

## 8. 외부망 필요 여부

| 단계 | 인터넷 필요? |
|------|------------|
| Ollama 설치 | ✅ 필요 |
| 모델 다운로드 | ✅ 필요 |
| **실제 AI 사용** | ❌ **불필요** |
| 모델 추가 다운로드 | ✅ 필요 |

모델을 한 번 받아놓으면 이후엔 **완전히 내 컴퓨터 안에서만 동작.**

---

## 9. 지식 컷오프 (Knowledge Cutoff)

모델은 **학습된 날짜 이후의 정보는 알 수 없음.**

```
Gemma3 학습 데이터가 2025년 초까지라면

✅ 알고 있는 것  → 2025년 초 이전 정보
❌ 모르는 것     → 2025년 초 이후 정보
```

### 최신 정보를 유지하려면?
- Google이 새 버전 출시 → `ollama pull gemma3` 로 업데이트
- 모델 학습은 비용/시간이 크므로 보통 **1년에 1~2번** 새 버전 출시

---

## 10. 실시간 웹 검색

| 환경 | 웹 검색 | 대안 |
|------|--------|------|
| 외부망 | ✅ 가능 | - |
| 내부망 | ❌ 불가능 | RAG (내부 문서 검색) |

### RAG란?
웹 대신 **내가 가진 문서들을 검색**해서 AI가 그걸 바탕으로 답변하게 하는 방식.

```
내부망 환경
├── AI 모델
└── RAG (내부 문서 검색) → 사내 문서, 매뉴얼, DB 등
```

---

## 한줄 요약

> Ollama는 **빈 플레이어**고, 모델은 **하나씩 따로 받는 것**이며, 받은 모델은 **오프라인으로 동작**하고, **학습된 날짜 이후 정보는 알 수 없다.**
> 최신 정보가 필요하면 모델 업데이트 또는 RAG/웹 검색을 활용한다.
